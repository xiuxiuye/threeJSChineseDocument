事件调度器（EventDispatcher）

自定义对象的JavaScript事件。

构造器（Constructor）

EventDispatcher()

创建一个 EventDispatcher 对象。

方法（Methods）

# .addEventListener ( type, listener )

type - 侦听的事件类型。
listener - 侦听器，即事件被触发时调用的函数。
为某类型事件添加侦听器。

# .hasEventListener ( type, listener )

type - 侦听的事件类型。
listener - 侦听器，即事件被触发时调用的函数。
检测某类型事件是否已添加侦听器。

# .removeEventListener ( type, listener )

type - 要删除的侦听器类型。
listener - 要删除的侦听器函数。
删除某类型事件的侦听器。

# .dispatchEvent ( event )

event - 被触发的事件。
触发某类型事件。
