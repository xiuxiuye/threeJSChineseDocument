立方体相机（CubeCamera）

创建6个远景相机并渲染到1个WebGL渲染器目标(WebGLRenderTarget)Cube对象上。

示例

materials / cubemap / dynamic
materials / cubemap / dynamic2
shading / physical
//创建立方体相机 var cubeCamera = new THREE.CubeCamera( 1, 100000, 128 ); scene.add( cubeCamera ); //创建汽车 var chromeMaterial = new THREE.MeshLambertMaterial( { color: 0xffffff, envMap: cubeCamera.renderTarget } ); var car = new Mesh( carGeometry, chromeMaterial ); scene.add( car ); //更新渲染目标立方体 car.setVisible( false ); cubeCamera.position.copy( car.position ); cubeCamera.updateCubeMap( renderer, scene ); //绘制场景 car.setVisible( true ); renderer.render( scene, camera );

构造器（Constructor）

CubeCamera( near, far, cubeResolution )

near -- 近裁剪面距离
far -- 远裁剪面距离
cubeResolution -- 设置立方体的宽度
构造1个包含6个PerspectiveCameras的立方体相机，然后渲染到1个WebGL渲染器立方体目标(WebGLRenderTargetCube)对象上。

属性（Properties）

# .renderTarget

生成的立方体纹理。

方法（Methods）

# .updateCubeMap ( renderer, scene )

renderer -- 当前的WebGL渲染器
scene -- 当前场景。
该方法用来更新渲染目标对象（renderTarget）。
